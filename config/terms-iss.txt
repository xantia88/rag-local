Параметр `Вид исследуемого объекта 1 уровень` используется для описания типа исследуемого объекта ИТ ландшафтра.
Параметр `Объект исследования` используется для описания исследуемого объекта ИТ ландшафта.
Параметр `Код вопроса` используется для описания уникального кода вопроса.
Параметр `Выявленное отклонение` используется для описания выявленного отклонения от требований ИТ стандарта.
Параметр `Скрытый комментарий по выявленному отклонению` используется для описания комментария по выявленному отклонению.
Параметр `Негативные последствия от проявления отклонения` используется для описания негативных последствий, связанных с выявленным отклонением.
Параметр `Рекомендации по устранению отклонения` используется для описания рекомендаций по устранению выявленного отклонения.
Параметр `Уровень отклонения` используется для описания уровня отклонения.
Параметр `Виды возможных рисков` используется для описания видов возможных рисков, связанных с выявленным отклонением.
