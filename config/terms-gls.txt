Параметр `Термин` используется для описания основного наименования термина.
Параметр `Алиас` используется для описания других возможных наименований термина, помимо основного наименования.
Параметр `Сокращение` используется описания сокращенного наименования термина.
Параметр `Определение` используется для описания смыслового значения термина.
Параметр `Используется в` используется для описания артефактов (документов) в которых используется данный термин.
